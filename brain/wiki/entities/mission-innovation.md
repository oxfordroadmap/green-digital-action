# Mission Innovation

## Basic Information
- Type: project
- Source: [[sources/itu-t-l-1480-greening-by-digital|itu-t-l-1480-greening-by-digital]]

## Description
Mission Innovation is a global initiative aiming to accelerate clean energy innovation through international collaboration between 23 countries and the European Commission. The initiative provides a framework for avoided emissions, which serves as a relevant methodological resource for investors and decision-makers. This framework assists in evaluating products and projects that contribute positively to the global [[concepts/climate-action|Climate Action]] and the broader decarbonization of society.
