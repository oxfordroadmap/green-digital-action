# Cancun Adaptation Framework

## Basic Information
- Type: project
- Source: [[sources/itu-t-l-1501-adaptation-best-practices|ITU-T L.1501 (Adaptation Best Practices)]]

## Description
The Cancun Adaptation Framework (CAF) is a global initiative established under the [[entities/unfccc|United Nations Framework Convention on Climate Change]] in 2010. It focuses on enhancing action on [[concepts/climate-action|climate change adaptation]] by enabling the planning, prioritization, and implementation of adaptation strategies. The framework emphasizes a country-driven, gender-sensitive, and participatory approach to building resilience against climatic risks. By facilitating international cooperation, it aims to reduce human vulnerability to the current global climate crisis.
