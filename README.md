# LLM-wiki-quarto-template

A proof-of-concept configuration for a two-tier knowledge repository.

- **Public Workspace (/):** Outward-facing research hub compiled via Quarto into a responsive web interface.
- **Private Workspace (/brain):** Isolated local wiki (Markdown/Obsidian) for raw notes and sensitive datasets, completely protected via .gitignore.

> [!WARNING]
> **Work-in-Progress (WIP)** > This framework configuration is actively undergoing structural testing. Please be aware of potential breaking changes, layout regressions, or configuration validation risks during updates. Use in production with appropriate care.

## Getting Started
1. Click **Use this template** to generate a clean copy.
2. Run quarto preview to start the local development server.
